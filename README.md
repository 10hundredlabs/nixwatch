# NixWatch

NixWatch is a terminal-inspired Wear OS watch face with bold, readable time and date, configurable date formats, colorful command-line styling, and glanceable complication rows for battery, steps, sun, and weather.

## Status

NixWatch is currently in closed testing for Google Play.

## Privacy

NixWatch does not collect, store, transmit, sell, or share personal data.
